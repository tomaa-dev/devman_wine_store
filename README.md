# Продаём элитное вино

Сайт магазина авторского вина "Новое русское вино". Небольшой скрипт на Python, который читает данные о винах из Excel-файла, подставляет их в Jinja2-шаблон и генерирует статическую страницу index.html, а затем запускает простой HTTP-сервер для локального просмотра.

## Как установить

Операционная система: любая, где доступен Python 3 (Windows, macOS, Linux). Используем 'pip' для установки зависимостей (requests, python-dotenv):
```
pip install -r requirements.txt
```
Рекомендуется использовать virtualenv/env для изоляции проекта.

## Запуск

- Скачайте код
- Запустите сайт командой:
```
python main.py
```
- Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).

Работа сервера будет выполняться в текущем терминале до прерывания (Ctrl+C)
